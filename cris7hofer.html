Guía Básica de Git

1. Inicializar un repositorio

Se usa para crear un repositorio Git en tu proyecto.

git init Aprender más

2. Ver estado del repositorio

Muestra los archivos modificados, añadidos o sin seguimiento.

git status

3. Agregar archivos

Agrega archivos al área de preparación (staging).

git add archivo.txt git add .

4. Crear un commit

Guarda los cambios en el historial del repositorio.

git commit -m "Mensaje del commit"

5. Crear una rama

Permite trabajar en nuevas funcionalidades sin afectar la principal.

git branch nombre-rama

6. Cambiar de rama (checkout)

Cambia entre ramas existentes.

git checkout nombre-rama

7. Crear y cambiar de rama

Crea una nueva rama y te mueve a ella directamente.

git checkout -b nombre-rama

8. Fusionar ramas (merge)

Une los cambios de una rama a otra.

git checkout main git merge nombre-rama

9. Ver historial

Muestra los commits realizados.

git log

10. Conectar a repositorio remoto

Conecta tu proyecto con GitHub u otro servidor.

git remote add origin URL

11. Subir cambios (push)

Envía los commits al repositorio remoto.

git push origin main

12. Descargar cambios (pull)

Trae y actualiza los cambios desde el repositorio remoto.

git pull origin main

Flujo básico de Git

Orden típico de trabajo:

git init
git add .
git commit -m "Primer commit"
git branch nueva-rama
git checkout nueva-rama
git merge nueva-rama
git push origin main
